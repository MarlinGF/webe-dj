
# ArtistLive

ArtistLive is a modern DJ software application built with Next.js, React, and Firebase. It provides a studio environment for internet radio broadcasters to manage libraries, create playlists, and perform live mixes with smooth crossfading.

## Features

- **Dual Deck Interface**: Independent controls for two audio decks.
- **Dynamic Crossfader**: Smooth transitions between tracks with adjustable fade speeds.
- **Continuous Play**: Automated track switching for non-stop broadcasting.
- **Track Library**: Organize and upload songs and commercials.
- **Playlist Management**: Create and manage custom sequences of tracks.
- **Cloud Powered**: Hosted on Firebase with Firestore for data and Storage for audio.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend**: [Firebase](https://firebase.google.com/) (Auth, Firestore, Storage)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your Firebase project and add your credentials to `.env.local`.
4. Run the development server: `npm run dev`
